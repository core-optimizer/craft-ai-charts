ðŸ“– Quick Start:
  1. Create a ModelSpec:
     kubectl apply -f - <<EOF
     apiVersion: craft.ai/v1alpha1
     kind: ModelSpec
     metadata:
       name: llama3-8b
     spec:
       category: llm
       provider: meta
       displayName: "Llama 3 8B Instruct"
       parameters: "8B"
       contextLength: 8192
       source:
         huggingFace: meta-llama/Meta-Llama-3-8B-Instruct
       serving:
         runtime: vllm
         port: 8000
       variants:
         - quantization: fp16
           memory:
             weights: "16Gi"
     EOF

  2. Create a ModelDeployment:
     kubectl apply -f - <<EOF
     apiVersion: craft.ai/v1alpha1
     kind: ModelDeployment
     metadata:
       name: llama3-8b-prod
       namespace: default
     spec:
       modelRef: llama3-8b
       quantization: fp16
       scaling:
         minReplicas: 1
         maxReplicas: 3
     EOF

  3. Check status:
     kubectl get modelspecs
     kubectl get modeldeployments -A
